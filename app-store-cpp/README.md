# App Store CPP Workspace

This folder contains all Custom Product Page (CPP) files for this branch.

## Quick Links

- Interactive preview: `app-store-cpp/preview/app-store-cpp-preview.html`
- Static screenshot reference: `app-store-cpp/preview/app-store-screenshots.html`
- Content data: `app-store-cpp/preview/cpp-data.js`
- Editing guide: `app-store-cpp/docs/CPP-EDITING-GUIDE.md`
- Creative guide: `app-store-cpp/docs/app-store-cpp-guide.md`

## Structure

- `preview/` - HTML preview tools and content data
- `docs/` - CPP guidance and specifications
- `assets/` - source assets used by preview files
- `exports/iphone/` - pre-rendered iPhone images
- `exports/tablet/` - pre-rendered tablet images

## Notes

- Legacy root links (`app-store-cpp-preview.html` and `app-store-screenshots.html`) now redirect into this folder.
- Open the preview via a local server for best export compatibility (`npx serve` or `python3 -m http.server`).
